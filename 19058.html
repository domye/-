Hexo快速搭建博客教程

Hexo

Hexo 是一个快速、简洁且高效的博客框架。支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。Node.js 所带来的超快生成速度，让上百个页面在几秒内瞬间完成渲染，只需一条指令即可部署到 GitHub Pages, Heroku 或其他平台。强大的 API 带来无限的可能，与数种模板引擎（EJS，Pug，Nunjucks）和工具（Babel，PostCSS，Less/Sass）轻易集成。

配置环境

安装Hexo前，要先安装几个程序。(安装方法自行查找，本文不再赘述)

  • Node.js
  • Git（镜像）

开始安装

打开d（其他也行）盘，打开终端（右键，git bash)，复制以下内容到终端中，回车。等待一段时间

npm install -g hexo-cli

建站

安装完 Hexo 后，开始建站

在终端中输入以下内容(#号及后面的不要)

mkdir Blog #创建Blog文件夹
hexo init #载入hexo
cd Blog #进入Blog文件夹
npm install #安装npm

其中，Hexo 是文件夹的名字，可以修改。

预览

在终端中输入：hexo server或者 hexo s

看到。。。。http://localhost:4000/。。。。

在浏览器中打开localhost:4000`即可预览。

（更多Hexo内容可以到Hexo官网学习，自行学习)

配置Hexo

打开Hexo目录下的_config.yml（记事本即可），找到并修改以下内容：

# Site
title: Blog         # 此处改为你站点的标题
subtitle:           # 此处改为你站点的副标题
description:        # 此处改为你站点的说明
keywords:           # 此处改为你站点的关键字
author:             # 此处改为你的名字
language:           # 此处改为 zh-CN
timezone:           # 此处改为 Asia/Shanghai


# URL
## If your site is put in a subdirectory, set url as 'http://yoursite.com/child' and root as '/child/'
url: http://yoursite.com
root: /
permalink: :year/:month/:day/:title/   # 此处可以改为 :title/
permalink_defaults:



# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: landscape   # 此处改为你主题名

修改完之后，再次在终端中输入：hexo clean和hexo s，然后在浏览器中打开localhost:4000，即可出现效果

配置主题

自行上各主题的官网学习，这里推荐butterfly主题，点击进去学习

连接 代码托管平台

国内推荐gitee和coding，国外推荐github，请自行注册好

右键 -> Git Bash Here，设置用户名和邮箱：

git config --global user.name "GitHub 用户名"
git config --global user.email "GitHub 邮箱"

创建 SSH 密匙：

输入 ssh-keygen -t rsa -C "GitHub 邮箱"，然后一路回车。

添加密匙：

进入 [C:\用户名\用户名.ssh] 目录（要勾选显示“隐藏的项目”），用记事本打开公钥 id_rsa.pub 文件并复制里面的内容。

登陆代码托管平台 ，进入 设置页面，选择左边栏的 SSH什么的，点进去。

随便取个名字，粘贴复制的 id_rsa.pub 内容到 Key 中，点击 添加。

创建仓库

coding这里推荐Coding官方文档，page服务在项目里点击『代码 -> Pages 服务』 …… 即可，所有代码托管平台操作方法都大同小异，自行研究

创建完项目之后，进入项目，点击克隆或者下载什么的，出现https什么的，点击下拉框，选择ssh，然后复制右边的

然后打开根目录的_config.yml文件，找到deploy：

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type:

并将它修改成如下

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: 
    coding: 刚复制的链接

然后在终端中输入hexo d -g，你的网站就上传到Coding了。其余代码托管平台也一样。
