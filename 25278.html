Windows快捷键大全

Windows

Windows 键：打开开始屏幕
Windows 键+空格键：切换输入语言和键盘布局
Windows 键+V：剪切板
Windows 键+Shift+. ：将应用移至左侧
Windows 键+I：打开设置栏
Windows 键+Q：打开应用搜索面板
Windows 键+Tab：循环切换应用
Windows 键+X：快捷菜单
Windows 键 + ←：最大化窗口到左侧的屏幕上(与 Metro 应用无关)
Windows 键 + →：最大化窗口到右侧的屏幕上(与 Metro 应用无关)
Windows 键+ ↑：最大化窗口(与 Metro 应用无关)
Windows 键+ ↓：最小化窗口(与 Metro 应用无关)
Windows 键+ SHIFT +↑：垂直拉伸窗口，宽度不变(与 Metro 应用无关)
Windows 键+ SHIFT +↓：垂直缩小窗口，宽度不变(与 Metro 应用无关)
Windows 键+SHIFT+←：将活动窗口移至左侧显示器 (与 Metro 应用无关)
Windows 键+SHIFT+→：将活动窗口移至右侧显示器(与 Metro 应用无关)
Windows 键+ 数字键：打开或切换位于任务栏指定位置的程序
Windows 键+Shift+s：快速截屏并且复制到剪切板
[编辑图片]
毕竟通常我们截图用不到编辑图片，那么要编辑图片该怎么办呢？
别慌，Windows早给我们准备好了。
Win+R，输入pbrush打开画图，Ctrl+V，图片就直接被粘贴了。
然后就能用画图编辑了！
Windows 键+D：显示桌面，第二次键击恢复桌面 (不恢复 Metro 应用)
Windows 键+E：打开资源管理器
Windows 键+L：锁住电脑或切换用户
Windows 键+M：最小化所有窗口
Windows 键+R：打开“运行”对话框
Windows 键+T：切换任务栏上的程序
Windows 键+U：打开轻松使用中心
Windows 键+S：打开windows自带搜索同windows键+Q
Windows 键+减号：缩小(放大镜)
Windows 键+加号：放大(放大镜)
Windows 键+Esc：关闭放大镜

Ctrl

Ctrl+鼠标滚轮：更改图标大小（桌面）、缩放（开始屏幕）
Ctrl+A：选择所有
Ctrl+C：复制
Ctrl+E：选择搜索框（资源管理器）
Ctrl+N：新建窗口
Ctrl+R：刷新
Ctrl+V：粘贴
Ctrl+W：关闭当前窗口（资源管理器）
Ctrl+X：剪切
Ctrl+Y：重做
Ctrl+X：撤销
Ctrl+Esc:开始屏幕
Ctrl+Tab：在 Metro 应用历史中循环切换
Ctrl+Ins：复制
Alt+D：选择地址栏（资源管理器）
Alt+Enter：属性
Alt+Spacebar：快捷菜单
Alt+Tab：应用程序间切换
Alt+Ins：粘贴
Ctrl+Shift+N：新建文件夹（资源管理器）
Ctrl+Shift+Esc：任务管理器
Ctrl+Shift+上下左右箭头：选择一段文字

网页

空格键 向下滚动网页。
Home 转至网页顶部。
End 转至网页底部。
Shift +鼠标滚轮 在网页上横向滚动。
Ctrl+D将当前网页保存为书签。
F11在全屏模式下打开网页。再按一次 F11 可退出全屏模式。
点击鼠标中键（滚轮键）。激活自动滚动模式。
Ctrl+U打开当前网页的源代码。
Ctrl++，Ctrl+-，按住 Ctrl 键并且向上/下滚动鼠标滚轮，放大/缩小网页上的所有内容。
Ctrl+0 将网页上的所有内容都恢复到正常大小。
Ctrl+F打开查找栏。
Ctrl+P 打印当前网页。
Ctrl+S保存当前网页。
F5 或 Ctrl+R重新载入当前网页。
Esc停止载入当前网页。
Ctrl+F5 或 Shift+F5重新载入当前网页，忽略缓存的内容。
