安卓端浏览器评鉴

仅供参考，具体使用情况因人&手机而异

大厂

QQ浏览器

  • 优点:
    独特的X5内核，视频播霸（缓存流畅，下载方便)，文件管理，解压缩，m3u3转MP4
  • 缺点:
  1. 默认搜索引擎是搜狗
  2. 云加速建议关闭，腾讯屏蔽了很多网站
  3. 无处不在的广告，每次更新都在添加广告。自家快报看点的横幅，微视的广告也会在看小电影的网站跳出来[受虐滑稽]，而且无法屏蔽，据说是X5内核里做了手脚。
  4. 资讯漫天，新闻分发使人恶心
  5. 推荐用修改版

UC浏览器

  • 优点:
  1. UC的内核也是独有的U3内核，视频控件也是自家开发的，网页拼接非常好用。其它和上面相似
  • 缺点:
  1. 臃肿不堪，广告爆满，是低端机的噩梦。比烂更烂，在此不再赘言
  2. 云加速建议关闭
  3. 资讯轰炸，UC震惊
  4. 总体不如QQ浏览器，搜索引擎默认神马，不可更改

360浏览器

  • 优点:
  1. 广告过滤，主页引擎自定义
  2. 功能完善，符合本人上网习惯
  3. 能够与电脑版 360安全浏览器同步密码
  4. 部分广告可关闭，网页广告可屏蔽，主页可自定义
  • 缺点:
  1. 开屏广告
  2. 应用内下载时候的广告
  3. 各种隐藏的推荐资讯
  4. 无书签搜索
  5. 无网页全屏翻译，搜索时显示书签已有网址
  6. 总体还可以，可以忍受一些功能缺失的朋友可以使用360极速浏览器，无广告，不比夸克差，不过已经停更

夸克浏览器

  • 优点:
  1. 界面比较简洁
  2. 去广告的能力强，和百度相比少了很多主页推送和营销号
  3. 支持多线程，该有的功能都有
  4. AI搜索越来越差
  • 缺点:
  1. 语音鸡肋，小说书架更是让人不爽，点开搜索框后响应时间慢
  2. 占用储存太大
  3. 图片难找，没有专门的图片选项
  4. 日渐臃肿，有走向UC老路的趋势，推荐用老版或修改版

谷歌浏览器

  • 优点:
  1. 稳定，启动迅捷流畅
  2. 网页设置人性化，操作简便
  3. 观看视频高速流畅，自带的外语翻译功能是所有浏览器中最强最准确的。
  4. UI简单美观，圆角看着舒服
  5. 断掉网络可以玩跳跳龙
  • 缺点:
  1. 国内不能云同步书签，用不了谷歌搜索引擎，可以更换
  2. 每次打开浏览器都会自动打开之前的网页
  3. 询问文件保存位置已打开，下载时还是自动下
  4. 默认没有底栏

火狐浏览器

  • 优点:
  1. 隐私保护
  2. 多终端同步，跨平台密码记录和自动填写。
  3. 可使用部分插件
  4. 流畅，兼容性强
  • 缺点:
  1. 国际版国内版不同，请选择Mozilla组织出品的Firefox，而非谋智中国 公司的。
  2. 操作不「先进」，但应该有相应插件可以搞

Edge浏览器

  • 优点:
  1. 自动同步密码和表单，以及收藏夹、集锦
  2. 谷歌内核
  3. 分为四个版本，稳定版，月更版，周更版，日更版
  • 缺点:
  1. bug多
  2. 网页打开速度极慢
  3. 矫枉过正的内容拦截器，图片杀手
  4. 恶心的选项卡手势设定（现在解决了不少）

Kiwi浏览器

  • 优点:
  1. 颜值高，启动迅捷，网页设置人性化
  2. 内置Yahoo、Google、Bing、DockDockGo等多个搜索引擎，搜索功能强大
  3. 观看视频高速流畅
  4. 可以安装和使用插件和扩展文件，并有较强的广告拦截能力
  • 缺点:
  1. 本身没有自带的外语翻译功能，需要安装彩云小译等翻译插件
  2. 更新慢

独立开发者

via浏览器

  • 优点:
  1. 安装包仅500kb，内存占用也极小
  2. 清流系的典范。可定制化很强，可以云同步
  3. 可以更换ua，可以添加脚本，广告规则可以导入
  4. 书签支持导入导出，支持资源嗅探。内置脚本网站（脚本资源一般）
  5. 支持网页翻译
  • 缺点:
  1. 内核调用系统内核，视频控件是原生，功能简陋，无文件管理，无阅读模式

X浏览器

  • 优点:
  1. 安装包内存和via差不多，内存占用也是
  2. 界面清爽，可定制化强，支持云同步
  3. 广告规则可自定义，浏览器ua可自定义，支持阅读模式，支持资源嗅探，即时翻译
  4. 支持书签导入导出【脚本功能最棒，可以直接安装油猴脚本!!!】
  5. 功能比via多，更加实用
  • 缺点:
  1. 调用系统内核（本APP另有X5内核版本）

米侠浏览器

  • 优点:
  1. 真正的视频嗅探，自定义界面和添加脚本
  2. 其余基本和via一致
  • 缺点:
  1. 缺少倍速播放
  2. 有时候点击网页会出现闪退
  3. 广告无法完全屏蔽。

神奇浏览器

  • 优点:
  1. 界面简洁漂亮，支持资源嗅探，自带订阅adblock plus规则的功能
  2. 划词翻译、阅读模式和看图模式用起来很爽。
  3. 其余基本和via一致
  • 缺点:
  1. 没有脚本插件

荟萃浏览器

  • 优点:
  1. 支持云同步，可自定义浏览器ua，可定制化强，可媒体嗅探，支持阅读模式，支持书签导入导出，看图模式，目录识别。支持整页翻译
  2. 内置视频播放器（体验一般），广告拦截自定义规则，内置脚本网站（脚本资源还算丰富）
  • 缺点:
  1. 文件管理功能鸡肋

雨见浏览器

  • 优点:
  1. 支持云同步，可自定义浏览器ua，浏览器核心可随时更换（支持X5内核和cw内核）
  2. 内置视频播放器，内置轻应用（网盘搜索，视频库解析等功能很有特色）
  3. 有侧边栏快捷方式和书签侧滑，广告规则自定义，内置脚本网站（资源一般），支持资源嗅探，网页翻译，自带语音助手。
  • 缺点:
  1. 高级功能需要开通会员（可看广告开通临时会员）
  2. 功能略显杂乱，流畅性有待测试
  3. 书签不能自由导出导入，文件管理缺失
